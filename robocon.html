Robocon 2010

Project Description

The project consisted of designing and fabricating a set of two autonomous robots and one manual robot which will act together to make a system of three pyramids of Egypt (Khufu, Khafraa and Mankaura). The manual robot was controlled using a PS3 (Playstation) Remote which was interfaced and customized to suit our requirements. The autonomous robots were controlled via the input from sensors and the programme code fed into them.

Robocon 2011

Project Description

We built a system of three robots - two autonomous and one manual in accordance with the Problem Statement. The three robots worked in synchronization with each other to collect a number of objects spread throughout the arena and assemble them to form a predefined assembly. The manual robot was controlled using a customized remote, while autonomous robots had plethora of sensors on it, which sensed the environment and took the action accordingly. The entire project was done indigenously by members of Team ROBOCON, IIT Roorkee - right from designing and fabricating the mechanical structure to the PCB designing, sensor calibration, coding and testing.

Robocon 2012

Project Description

The THEME for ROBOCON 2012 was “Peng On Dai Gat”, meaning peace and prosperity. A manual robot and two autonomous robots were to be fabricated with given weight and size constraints. Cylindrical "buns" were to be collected from variable heights upto 1.5 metres and were to be put into a basket brought by manual robot. Fabrication of PCBs was done using Eagle 5.10 for ARDUINO Mega2560 and power supply was done. Magnetometers (digital Compass I2C communication) were calibrated and tested to use for turning of autonomous robots.

Robocon 2013

Project Description

The THEME for ROBOCON 2013 was GREEN PLANET. A team is declared winner if it can complete the task by throwing bud on moon. Each team had to design 2 Robots, 1 manual and 1 autonomous. The manual robot was designed such to pick seven leaves using pneumatically controlled grippers, place three in the manual zone and carry the rest four leaves to the automatic zone for the Automatic Robot. According to our strategy, the function of the automatic robot was to place 4 leaves on the arena. Two leaves in common zone and two in the Team zone. Biggest challenge before us was to successfully test the encoders for the locomotion of the robot. Our electronics Team successfully traced the path decided using magnetometer and encoder. Use of line sensors was done for placing the leaves exactly in the ring.

Robocon 2014

Project Description

The THEME for ROBOCON 2014 is "A SALUTE TO PARENTHOOD". "The Objective" Two teams play various game activities in the park with their Parent robot and Child robot. The Parent robot has to carry Child robot up-to the play zone and the Child robot plays game activity . Each team plays three game activities such as Seesaw, Pole walk and Swing. Once the Child Robot completes three activities, it can challenge the fourth game activity, which is set in the middle of the park.The team successfully completing the fourth game activity first; accomplishes "SHABAASH" (appreciation) and is declared "The Winner". If no team is able to perform above mentioned winning task, the winner will be decided as per the stated rules.

Robocon 2015

Project Description

The THEME for ROBOCON 2015 is "ROBOMINTON". "The Objective" The motive of this contest theme is badminton's doubles game. The highlight of this game is how the two robots hit and hit back shuttle by collaborating each other. The longer the rally continues the more exciting the game becomes. The robot with unique way of hitting shuttle can be entertaining. The audience will be enthralled if the robot made an eye-opening jumping smash. We are looking forward to witnessing exciting matches of unique robots built by the young budding engineers in Pune, INDIA

Robocon 2016

Project Description

CONTEST THEME: Clean Energy Recharging the World
The game of ABU Robocon 2016 is designed in order to create the awareness of efficient energy consumption and clean and renewable energy utilization. Each team has to build two robots; Eco Robot and Hybrid Robot. Eco Robot doesn’t have an actuator to drive. It receives the driving energy from Hybrid Robot. Eco Robot has to use only one steering actuator to control its direction, to track the path containing Slopes and Hills, River, and Down Hill. Besides providing driving energy to Eco Robot, Hybrid Robot has to take Wind Turbine Propeller from Eco Robot and climb up Wind Turbine Pole in order to assemble Wind Turbine.

Robocon 2017

Project Description

CONTEST THEME: Asobi - The Landing Disc
The theme revolves around the word "asobi" (play), which is also a fundamental philosophy behind Robocon. In "asobi," playful, unique, original show of skills is often more important than winning or losing, as everyone - friend and foe alike - can applaud and enjoy them.
